🏆 Flutter Badge App

A simple **Flutter app** that gives badges to users when they complete certain achievements. The app uses **Provider** for state management and **flutter_badges** to display badges.


/badge_app
│── /lib               # Main code folder
│   │── main.dart      # Entry point of the app
│   │── /models        # Holds data models
│   │   ├── achievement.dart  # Defines the Achievement class
│   │── /providers     # Handles state management using Provider
│   │   ├── achievement_provider.dart
│   │── /screens       # Different screens of the app
│   │   ├── home_screen.dart  # Main screen with achievements list
│   │   ├── badge_screen.dart  # Displays earned badges
│   │── /widgets       # Reusable UI components
│   │   ├── badge_widget.dart  # Custom widget for badges
│── pubspec.yaml       # Project dependencies and configurations
│── README.md          # Project documentation



## **📜 How It Works**

1️⃣ The app starts with `main.dart`, which loads the `HomeScreen`.
2️⃣ **HomeScreen** displays a list of achievements and allows users to complete them.
3️⃣ When an achievement is completed, it is stored in `AchievementProvider`.
4️⃣ Completed achievements appear as badges on the  **BadgeScreen** .
5️⃣ The `flutter_badges` package is used to display the badges in a grid format.

class AchievementProvider with ChangeNotifier {

    Achievement(title: "5 Logins", description: "Logged in 5 times"),

  List<Achievement> get achievements => _achievements.where((ach) => ach.isCompleted).toList();

  void completeAchievement(String title) {
    for (var ach in _achievements) {

        ach.isCompleted = true;
        notifyListeners();
      }
    }
  }
